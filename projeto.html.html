HTML Linguagem de Marcação de Hipertexto

Tutorial para Iniciantes

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento de uma página da web (JavaScript).

Tabela

Objeto Definição

HTML

 É uma linguagem de marcação de hipertexto que define a estrutura e o significado do conteúdo de uma página web

HTTP

 É a sigla para Hypertext Transfer Protocol, ou Protocolo de Transferência de Hipertexto

CSS

 É uma linguagem de estilo que define como os elementos HTML serão apresentados no navegador

JavaScript

 É uma linguagem de programação que permite criar páginas web interativas e dinâmicas

PHP

 É uma linguagem de script usada para desenvolvimento web do lado do servidor

SQL

 É uma linguagem padrão para gerenciar e manipular bancos de dados relacionais

API

 Interface de Programação de Aplicações, um conjunto de protocolos para integração entre sistemas

Listas Ordenadas

  1. HTTP: é a sigla para Hypertext Transfer Protocol, ou Protocolo de Transferência de Hipertexto.
  2. TCP/IP: (Transmission Control Protocol) e IP (Internet Protocol) é um conjunto de protocolos de comunicação entre computadores que estão ligados em rede.
  3. FTP, SFTP e FTPS: são protocolos de transferência de arquivos que se diferenciam na segurança e na forma como trocam dados.
  4. POP, IMAP e SMTP: são protocolos de internet utilizados para o envio e recebimento de e-mails.
  5. DHCP: (Dynamic Host Configuration Protocol) é um protocolo de cliente/servidor que fornece automaticamente um host IP (Protocolo de Internet) com seu endereço IP e outras informações de configuração relacionadas, como a máscara de sub-rede e o gateway padrão.
